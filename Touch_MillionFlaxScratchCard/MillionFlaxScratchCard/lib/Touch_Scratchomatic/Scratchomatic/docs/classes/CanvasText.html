CanvasText Class

Defined in: Scratchomatic/scripts/uncompressed/CanvasText.js:1
Module: CanvasText

Describes a factory object that writes formatted text to an offscreen canvas element using fillText.

  • Can then be used as an image source for an Actor or written directly to an onscreen canvas element.
  • Gives a performance improvement over using fillText every frame.
  • Includes functions for font preload detection.

    • [fontFamily= "arial"] String optional

      The font family to use

    • [fontSize= "10px"] String optional

      Text size / line height

    • [fontStyle= "normal"] String optional

      normal|italic|oblique

    • [fontWeight= "normal"] String optional

      normal|bold|bolder|lighter|100|200...

    • [fontVariant= "normal"] String optional

      normal|small-caps

    • [fillStyle= "black"] String optional

      Fill colour to apply to the characters

    • [strokeStyle] String optional

      Fill colour to apply to stroke (i.e. text outline)

    • [strokeStyles= []] Array optional

      Fill colours to apply to strokes (i.e. text outlines). Listed in drawing order.

    • [lineWidth] String optional

      Stroke width in pixels.

    • [lineWidths= []] Array optional

      Stroke widths in pixels. Listed in drawing order.

    • [shadowColour= null] String optional

      Fill colour to apply to text shadow. Null = no shadow.

    • [shadowBlur= 0] Number optional

      Gaussian shadow blur radius in px. Larger numbers = more blur.

    • [shadowOffsetX= 0] Number optional

      Horizontal distance between shadow and characters in px.

    • [shadowOffsetY= 0] Number optional

      Vertical distance between shadow and characters in px.

    • [baseHeightString= "WWW"] String optional

      A canvas element that is taller than any character must be created to measure the text height of a new CanvasText object. This is calculated by measuring the width of the baseHeightString setting and using that as the height. Normally this value never needs to be changed.

    • [measureHeightString= "IQqgjiy|0123456789,"] String optional

      This is the string that is written to the base canvas and used to measure the text height. It usually doesn't need to be changed unless a font has a particularly tall character or a game uses a reduced character set and you want to save some space.

    • [location] HTMLElement optional

      The place in the document that generated text will be placed. Useful when converting values from px.
