Defined in Scratchomatic/scripts/uncompressed/CanvasToolkit.js:66

Uses a cubic bezier curve to create a transition easing function.

  • Based on Firefox's nsSMILKeySpline.cpp
  • Easing concepts from here.
  • Code 'borrowed' from here.

Parameters:

  • mX1 Number

    X-Coordinate of the first handle point. Must be between 0 and 1.

  • mY1 Number

    Y-Coordinate of the first handle point. Must be between 0 and 1.

  • mX2 Number

    X-Coordinate of the second handle point. Must be between 0 and 1.

  • mY2 Number

    Y-Coordinate of the second handle point. Must be between 0 and 1.

Returns:

Function:

A cubic bezier curve function of the type function(progress), where progress is a value between 0 and 1.

Example:

 var spline = BezierEasing(0.25, 0.1, 0.25, 1.0);

spline(x) => returns the easing value | x must be in [0, 1] range
