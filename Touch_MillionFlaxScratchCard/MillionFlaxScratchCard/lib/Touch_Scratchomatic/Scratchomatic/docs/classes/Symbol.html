Symbol Class

Defined in: Scratchomatic/scripts/uncompressed/Symbol.js:15
Module: CanvasToolkit

A class that defines the unique aspects of a symbol that appears on a scratch card.

Constructor

Symbol

(
  • config
)

Defined in Scratchomatic/scripts/uncompressed/Symbol.js:15

Parameters:

  • config Object
    • group SymbolGroup

      Reference to the SymbolGroup that created this symbol

    • [texMult] Number optional

      Local copy of the texture multiplier.

    • [x] Number optional

      X-Coordinate in the group.

    • [y] Number optional

      Y-Coordinate in the group.

    • [width] Number optional

      Width of the symbol in px.

    • [height] Number optional

      Height of the symbol in px.

    • [backgroundObj] PGCImage optional

      Image used to draw the back of the symbol.

    • [value] String optional

      Symbol value as a string.

    • [valueObj] PGCImage optional

      Symbol value as a text image.

    • [prize] String optional

      Symbol prize as a string.

    • [prizeObj] PGCImage optional

      Symbol prize as a text image.

    • [specialObjs] Array optional

      Any additional PGCImage objects used by the symbol (e.g. particles, animated content)

    • [foilImage] PGCImage optional

      The canvas used to draw the foil for this symbol, wrapped in a PGCImage.

    • [foilContext] CanvasRenderingContext2d optional

      Reference to the 2d context of the foilImage's canvas.

    • [scratchSprite] PGCImage optional

      The sprite used to automatically scratch the symbol.

    • [scratchBBox] X: Number, y: Number, w: Number, h: Number optional

      } The area of the symbol that must be uncovered for this symbol to count as scratched.

    • [prizeSymbol= false] Boolean optional

      Flag indicating that this symbol is a prize symbol

animatePart

(
  • config
)

Defined in Scratchomatic/scripts/uncompressed/Symbol.js:650

Helper function. Gets a part of the symbol by name, and, if it's animated, performs the run operation on it.

Parameters:

  • config Object
    • [partName] String optional

      name of the symbol part to animate. This can be "value", "prize", "background", "foil"; or the name of a special element.

    • [partReference] PGCImage optional

      Direct reference to the symbol part to animate.

    • [visibleAfter= true] Boolean optional

      Sets the "visible" property on the part after the animation completes. Useful for parts that disappear after animating (e.g. particles)

    • [count= 1] Number optional

      Number of times to run the animation
